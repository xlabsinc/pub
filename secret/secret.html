🔐 Secret Encrypt/Decrypt

Purpose: Securely share API keys and secrets via text message.

Encrypt: Converts your secret to xxd hex format using AES-256-ECB encryption (equivalent to: echo "secret" | openssl aes-256-ecb -a -pass "pass:<pass>" -e | xxd -u | cut -c1-50)

Decrypt: Reverses the process to recover the original secret (equivalent to: cat out.txt | xxd -r | openssl aes-256-ecb -a -pass "pass:<pass>" -d)

✓ Fully compatible: Encrypt in UI → decrypt in shell, or encrypt in shell → decrypt in UI. Uses OpenSSL-compatible AES-256-ECB with salted key derivation (EVP_BytesToKey, SHA-256).

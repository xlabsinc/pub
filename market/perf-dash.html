      How to Use This Dashboard

      1. Load data first from Manage Data Sources (Add Url, Add local file, or Load Default Config).

      2. Click category chips to switch groups; double-click a category chip to rename it; use + to add a custom category.

      3. Use Add Symbols next to Category to append comma-separated symbols to the active category.

      4. Data is cached in-memory while this page is open, and background prefetch warms uncached symbols.

      Buttons and actions:

      - -5: show five fewer rows (minimum 5).

      - +5: show five more rows.

      - All: show all symbols in the active category.

      - Refresh: fetch only symbols that are missing key data.

      - Force Refresh: fetch all symbols again from source endpoints.

      - Download Config: download current categories/symbols as JSON.

      - Save Config: save current config to browser cookie.

      - Clear Config: clear saved cookie config.

      - Load Default Config: load the built-in default portfolio URL.

      - Add Url: add a remote HTTP/HTTPS JSON source.

      - Add local file: import a local JSON file.

      - Add Symbols: add comma-separated symbols to selected category.

      - Category x: remove that category from current view.

      - Row x: remove that symbol row from the active category view.

      - Row refresh icon (↻): refresh only that symbol.

      - ? in category strip: open this help.

      - + in category strip: add a new custom category.

Gist Explorer Help

Usernames: Enter one or more GitHub usernames, separated by commas.

Gist Type: Select the type of gists to fetch.

  • Public: Only public gists.
  • Secret: Only secret gists (requires a Personal Access Token).
  • All: Both public and secret gists (requires a Personal Access Token).

URL Hash Parameter: You can load users directly via the URL.
Example: gist.html#nx=user1,user2
